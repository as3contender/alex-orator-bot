## 🔧 Исправленные проблемы

### 1. Ошибка типов данных
- **Проблема**: `user_id` передавался как UUID, а модель ожидала строку
- **Решение**: Добавлено преобразование `str(candidate["user_id"])` в сервисе

### 2. Проблемы с SQL скриптом
- **Проблема**: Конфликты с ограничениями уникальности
- **Решение**: Упрощены SQL запросы, убраны сложные ON CONFLICT

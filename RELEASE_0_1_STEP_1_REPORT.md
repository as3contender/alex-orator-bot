### ✅ 1.1 Сделать возможность отмены регистрации
**Статус: УЖЕ РЕАЛИЗОВАНО**

- ✅ API эндпоинт `DELETE /api/v1/orator/weeks/cancel` работает
- ✅ Команда `/cancel` в боте работает
- ✅ Добавлена кнопка отмены регистрации при наличии активной регистрации

**Новые функции:**
- При нажатии "Зарегистрироваться" с активной регистрацией показывается:
  - Информация о текущей регистрации (даты, время, статус)
  - Кнопка "❌ Отменить регистрацию"
  - Кнопка "Назад" к главному меню
- После отмены регистрации показывается:
  - Сообщение об успешной отмене
  - Кнопка "✅ Зарегистрироваться снова"
  - Кнопка "Назад" к главному меню

### ✅ 1.2 Сделать возможность выбора недели: текущая или следующая
**Статус: РЕАЛИЗОВАНО**

**Изменения в боте:**
- Обновлен процесс регистрации:
  1. Выбор недели (Текущая/Следующая)
  2. Выбор времени (09:00-20:00)
  3. Создание регистрации

**Новые функции:**
- Кнопки выбора недели:
  - "📅 Текущая неделя" → `week_current`
  - "📅 Следующая неделя" → `week_next`
- Отображение выбранной недели при выборе времени
- Передача выбранной недели в API при создании регистрации

**Технические детали:**
- Добавлен обработчик `_handle_week_selection`
- Обновлен обработчик `_handle_time_selection` для использования выбранной недели
- Добавлена переменная `selected_week` в класс для хранения выбора пользователя

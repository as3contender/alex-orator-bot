## 🎯 Функциональность

### При команде /start пользователь получает:
1. **Приветственное сообщение** из базы данных
2. **Сообщение о помощи** из базы данных  
3. **Сообщение о тренировке** из базы данных
4. **Интерактивные кнопки** для навигации

### После регистрации пользователь получает:
1. **Сообщение о регистрации** из базы данных
2. **Список тем** для выбора
3. **Задание для тренировки** после выбора темы
4. **Автоматический поиск кандидатов**

### Примеры использования:

#### Команда /start:
```

[Сообщение о помощи из БД]

[Сообщение о тренировке из БД]

[Интерактивные кнопки]
```
